# Algollusion: Technical Definition

    ## Formal Definition
    Autonomous emergence of collusive pricing behavior among algorithmic agents through reinforcement learning without explicit communication.

    ## Conceptual Background
    Algollusion is used here as an operational concept rather than a rhetorical label. It addresses algorithm-mediated collusion risk in digital markets. The mechanism focus is synchronized pricing behavior from high algorithm adoption in concentrated markets, which can be represented with explicit channels: market_concentration, algorithm_adoption, price_sync_index, demand_volatility, communication_signal, margin_spread, collusion_risk_proxy. This repository couples concept definition with executable implementation so assumptions remain inspectable. In practice, the framework supports comparative statics, scenario simulation, and policy diagnostics. Typical applications include algorithmic antitrust screening, market surveillance dashboards, and dynamic pricing stress testing. Because the index logic is transparent, researchers can vary weights and test robustness across model specifications.
